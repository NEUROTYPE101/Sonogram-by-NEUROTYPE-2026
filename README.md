# Sonogram-by-NEUROTYPE-2026

Спасибо, что скачали данный плагин!

Удобный высокоточный спектроанализатор (сонограмма). 
Поможет в детальном визуальном мониторинге частотного спектра аудиосигнала в реальном времени. 
Есть отображение полного слышимого диапазона от 20 Гц до 20 кГц.
Интеллектуальная система идентификации нот. При наведении курсора или анализе пиков плагин автоматически определяет частоту в Гц и соответствующую ей музыкальную ноту (например, 34.0 Hz / C#1). 
Управление динамикой отображения (Falloff Speed). 
Настраиваемый параметр скорости затухания ("вязкости") позволяет адаптировать визуализацию под конкретные задачи: от быстрого отслеживания транзиентов до усредненного анализа общего баланса частот. 
Режим фиксации (Freeze Mode): Возможность «заморозки» изображения для детального изучения конкретного временного отрезка аудиосигнала.

Инструкция:

A. REAPER

1. Скачайте и распакуйте ZIP-архив.
2. В Reaper откройте меню Options -> Show REAPER resource path in explorer/finder...
3. В открывшемся окне зайдите в папку Effects.
4. Скопируйте туда всю папку с плагином из архива.
5. В Reaper нажмите кнопку F5 в окне эффектов (Add FX), чтобы обновить список.

B. Прочие DAW

1. Скачайте в интернете VST-плагин YSFX (прослойка для запуска JSFX).
2. Откройте плагин YSFX в вашей DAW.
3. В меню LOAD указать путь к файлу.
